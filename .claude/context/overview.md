# Mama — overview

A React prototype for **Mama**, a mom-friendship matching app.

## Positioning

- **Anti-Tinder.** The core value is *real meetups*, not endless chat.
- **Verified-only.** Slows signup intentionally — it's the moat.
- **Editorial-warm aesthetic.** Magazine cover energy, terracotta and sage palette, Fraunces serif headlines.

## Stack

- Vite + React 18
- Tailwind CSS (via PostCSS)
- `lucide-react` for icons
- No test framework yet
- Single-file prototype: `src/App.jsx` (~4,000 lines)

## Scripts

| Command | Purpose |
|---|---|
| `npm run dev` | Vite dev server |
| `npm run build` | Production build |
| `npm run preview` | Preview the built app |

## Target audience

Moms looking for real-life friendship with other moms — not chat pen-pals, not dating-app-style swiping. The app is built to feel like a curated magazine, not a feed.
